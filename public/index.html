my projects

here are a couple of projects i am really proud of:

  • collab-notepad: i wrote an online collaborative text editor in elixir and svelte, for which i wrote and tested my own conflict resolution algorithm that prevented edit conflicts. the result of this is that editors can pull changes from the server without pushing, and vice versa. github repository
  • qart: i figured out how to make russ cox's qart codes, and wrote a rust command line application that makes them. i published this as a crate on crates.io, which has close to 1000 downloads. github repository crate
  • md-sticky: i found myself disliking the editor of the default sticky notes app that comes with macos, and rather than pay for one i liked i decided to make my own with tauri (rust & svelte). it is now something i use daily, while i iron out bugs as i find them. github repository
